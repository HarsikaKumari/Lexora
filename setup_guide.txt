# LexiLink | Legal Consultation Platform - Local Setup Guide

Follow these steps to run the LexiLink project on your local machine:

## 1. Prerequisites
Ensure you have the following installed:
- Node.js (v18 or higher)
- npm or yarn

## 2. Initial Setup
1. Open your terminal or command prompt.
2. Navigate to the project directory:
   cd path/to/your/project
3. Install dependencies:
   npm install

## 3. Firebase Configuration (Optional but Recommended)
To enable real Authentication and Firestore features:
1. Go to the Firebase Console (https://console.firebase.google.com/).
2. Create a new project named "LexiLink".
3. Add a "Web App" to the project.
4. Copy the `firebaseConfig` object provided by Firebase.
5. In the project directory, find `src/firebase/config.js`.
6. Replace the placeholder values in the `firebaseConfig` object with your actual keys.

## 4. Running the Development Server
1. Start the Vite development server:
   npm run dev
2. The application should now be running. Open your browser and navigate to:
   http://localhost:3000

## 5. Building for Production
To create a production-ready bundle:
   npm run build

## 6. Project Structure Overview
- /src/components: Shared UI components (Layout, Hero, Features).
- /src/features/auth: Registration and login logic for clients and lawyers.
- /src/features/discovery: Lawyer search and smart filtering.
- /src/features/documents: Form-based PDF generation logic.
- /src/features/consultation: Video/Audio consultation UI mock.
- /src/features/admin: Administrative validation and auditing tools.

## 7. Troubleshooting
- If "node" is not recognized: Ensure Node.js is added to your system's PATH.
- If dependencies fail: Try running `npm cache clean --force` and then `npm install`.

For further assistance, refer to the documentation or contact the administrator.
